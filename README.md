# ABES Hall Booking System

A modern web application for managing seminar hall and auditorium bookings at ABES Engineering College.

## Features

- **User Authentication** - Secure login/signup with role-based access (Admin, Faculty, Student)
- **Hall Booking** - Easy booking interface with date/time selection
- **Booking Management** - View, edit, and cancel bookings
- **Admin Dashboard** - Comprehensive admin panel for managing all bookings and users
- **Notifications** - Real-time notifications for booking status updates
- **User Profiles** - Manage personal information and preferences
- **Conflict Detection** - Automatic detection of booking conflicts
- **Recurring Bookings** - Schedule recurring events
- **Waiting List** - Join waiting list when halls are unavailable

## Tech Stack

- React 19 with Vite
- Tailwind CSS for styling
- Framer Motion for animations
- Lucide React for icons
- LocalStorage for data persistence

## Installation

```bash
# Clone the repository
git clone https://github.com/ashurauza/Seminar_Auditorium_Allocation_App.git

# Navigate to project directory
cd Seminar_Auditorium_Allocation_App

# Install dependencies
npm install

# Start development server
npm run dev
```

## Usage

1. Open your browser and navigate to `http://localhost:5173`
2. Register a new account or use existing credentials
3. Browse available halls and make bookings
4. Admin users can manage all bookings and users through the admin dashboard
